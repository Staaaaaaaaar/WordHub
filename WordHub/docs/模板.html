Learner 类说明

Learner 是 WordHub 的核心业务逻辑类，采用单例模式设计，确保全局只有一个用户实例在运行。

主要职责

  • 管理当前登录用户的状态信息。
  • 负责处理学习和复习的核心逻辑，是连接 UI 和数据库的桥梁。
  • 跟踪用户学习进度，判断并触发成就解锁事件。

关键方法


    // 获取 Learner 的全局唯一实例
    static Learner* getInstance();

    // 用户登录
    bool login(const QString& username, const QString& password);

    // 获取下一个需要学习的单词
    Word getNextWordToLearn();
