核心数据结构文档 (utils.h)

负责人: 刘继轩

一、概述

utils.h 文件定义了整个项目中使用到的核心数据实体结构。这些结构体和类作为数据传输对象（DTO），在程序的不同模块之间传递信息。

二、Word 类

单词类，存储一个单词的完整信息，是系统中最核心的数据实体。

属性

  • int id: 单词在数据库中的唯一标识。
  • QString word: 单词文本。
  • QVector<Phonetic> phonetics: 单词的音标列表。
  • QMap<QString, QVector<Definition>> meanings: 按词性分类的释义。
  • QDateTime lastReviewed: 最后复习时间。
  • int reviewCount: 复习次数。
  • int difficulty: 难度等级（1-5）。

三、Category 类

单词分类（词库）类。

属性

  • int id: 分类在数据库中的唯一标识。
  • QString name: 分类名称。
  • QString description: 对该分类的详细描述。

四、LearningRecord 类

学习记录类，记录用户的每一次学习行为。

属性

  • int id: 记录在数据库中的唯一标识。
  • int wordId: 关联的单词 ID。
  • int userId: 关联的用户 ID。
  • QDateTime timestamp: 本次学习发生的时间戳。
  • bool correct: 本次学习回答是否正确。
  • int difficulty: 学习时单词的难度。
